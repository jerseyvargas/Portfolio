Bear

Video Game Solo Project

Languages: HTML, CSS, Javascript

Bear is an HTML web-based video game where Bear Cub, played by the user, must navigate his way out of the burning forest. However, the entities within this forest will not make it easy! I was inspired by Limbo, a game critically acclaimed for its graphical elements and design. I wanted to emulate a similar ambient and eerie feel to my game. All assets including character design, character animations, and background renderings were created by me.

Data Visualization Project

Languages: HTML, CSS, Javascript

For this project, I was challenged to reinterpret a found data set into a responsive website incorporating qualities of the Dada art movement. I took two data sets from data.worldbank.org regarding the number of male and female adolescents in Mexico who have dropped out of school and visualized the data in a creative manner. My goal for this project was to ultimately raise awareness regarding education disparities.
